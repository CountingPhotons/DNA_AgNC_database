# DNA-AgNC Database

A browsable, read-only catalog of DNA-stabilized silver nanocluster
(DNA-AgNC) characterization data — absorption/emission spectra, quantum
yield, lifetime, cluster composition, DNA sequence, and reference images.

**🔗 Live site:** [view it here](https://countingphotons.github.io/DNA_AgNC_database/)

## What's in here

- `index.html`, `library.html`, `material.html`, `spectra.html` — the
  site pages
- `data/` — the full dataset as plain JSON (one combined catalog, plus
  one file per cluster)
- `static/` — images, fonts, and the JS libraries used to render the
  charts (all hosted locally — the site makes no calls to any external
  service)

## On this page

- 🔍 Search and sort the full cluster library
- 📈 View absorption/emission spectra with interactive hover readouts
- 🖼️ Browse reference images per cluster
- ⬇️ Download any cluster's spectra (CSV/ZIP) or full record (JSON)
  directly from your browser

## About this repository

This is a **static, read-only mirror** of our internal database — there
is no server, no database engine, and no way to submit, edit, or delete
entries through this site. It is rebuilt and re-published periodically
from the working database maintained by the project team.

**Spot an error, or have data to contribute?** Please contact
Simon Lind (simon.lind@chem.ku.dk) or Tom Vosch (Tom@chem.ku.dk).
